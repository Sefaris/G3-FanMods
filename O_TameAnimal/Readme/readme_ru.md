Требования: Main

Добавляет возможность приручать животных после изучения новых перков. Прирученное животное нейтрально к другим НПС. Его можно кормить и тем самым восполнять ему здоровье.

При изучении навыка "Вожак стаи" количество животных, которых можно приручить становится неограниченным.

Кабаны и болотные шныги больше не бросаются на игрока без предупреждения.

Варант наполнен газелями из мода LTM - автор Yuko.

В одной из пещер появилось несколько новых мобов - автор Adamtech.
