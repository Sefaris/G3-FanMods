Voraussetzungen: Main

Fuegt die Faehigkeit hinzu, Tiere zu zähmen, nachdem neue Perks erlernt wurden. 
Ein gezaehmtes Tier ist anderen NPCs gegenueber neutral. 
Es kann gefuettert werden und so seine Gesundheit wieder auffuellen.

Beim Erlernen der Fertigkeit "Rudelfuehrer" wird die Anzahl der Tiere, die gezaehmt werden koennen, unbegrenzt.
Wildschweine stuerzen sich nicht mehr ohne Vorwarnung auf den Spieler.
Die Variante ist mit Gazellen aus der LTM-Mod gefüllt - von Yuko.
Mehrere neue Mobs sind in einer der Hoehlen erschienen - von Adamtech.
