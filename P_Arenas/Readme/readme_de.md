Ein Update der Gladiatorenkarriere des Helden. 
Fast alle Arenen in dieser Mod wurden auf die, eine oder andere Weise veraendert. 
Die Kaempfer haben eine vielfaeltigere Ausruestung und Waffen.
In einzelnen Faellen wurden die Arenaregeln geaendert oder neue Quests hinzugefuegt.
Die Kämpfe in der großen Arena von Faring werden sich wie der Hoehepunkt aller Arena-Abenteuer des Helden anfuehlen, bei denen er sich an all die Erfahrungen erinnern muss, die er zuvor gesammelt hat.
