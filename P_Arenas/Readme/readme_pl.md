Wymagane: Main

Rozszerzenie kariery gladiatorskiej bohatera. Prawie wszystkie areny w tym modzie są w jakiś sposób zmienione. Wojownicy dysponują bardziej zróżnicowanym wyposażeniem. W niektórych przypadkach zmieniono zasady walki na arenie lub dodano nowe zadania. Bitwa na dużej arenie Faring będzie sprawiała wrażenie zwieńczenia wszystkich przygód bohatera na arenie, gdzie będzie musiał przypomnieć sobie wszystkie zdobyte wcześniej doświadczenia.
