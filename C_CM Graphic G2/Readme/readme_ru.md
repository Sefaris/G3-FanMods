Требования: Content Mod Base

Данная модификация добавляет графические изменения КонтентМода на манер Готики 2.

Изменения:
- Новый внешний вид доспехов паладина и гвардейца;
- Некоторые нпс будут одеты в доспехи из панцирей ползунов, в броню ополченцев и рыцарей;
- Ретекстур щита рыцаря и щита паладина;
- Повешенные тела паладинов в орочьих городах заменены на тела ополченцев;
- Новая внешность нежити в Готе;
- Старые знакомые из Готики 2 наряжены в свои узнаваемые доспехи (Лестер, Ларес, Ли, Горн, Диего, Ангар, Торус, Ксардас);
- Некоторые бандиты одеты в бандитскую одежду из Готики 2.
