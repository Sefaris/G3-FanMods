Wymagane: Content Mod Base

Ta modyfikacja dodaje zmiany graficzne do ContentModa w stylu Gothic 2.

Zmiany:
-Nowy wygląd zbroi paladyna i gwardzisty;
-Niektórzy NPC będą nosić zbroje z pełzaczy, milicji i rycerzy;
-Mieszanka tarcz rycerza i tarcz paladyna;
-Ciała powieszonych paladynów w miastach orków zastąpione ciałami milicji;
-Nowe wyglądy nieumarłych w Gocie;
-Starzy znajomi z Gothic 2 są ubrani w swoje zbroje (Lester, Lares, Lee, Gorn, Diego, Hangar, Thorus, Xardas);
-Niektórzy bandyci są ubrani w stroje bandytów z Gothic 2.
