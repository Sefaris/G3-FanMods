Требования: Main, UpdatePack 1.04 (1.05), ParallelUniversePatch

Данная модификация имеет сюжетные несоответствия с модом CM End Game.

Данная модификация полностью перерабатывает пиратскую линейку квестов из ЛТМ, превращая её в полноценную сюжетную линию в пиратском лагере.
Игрок должен заслужить право вступить в команду, выполнять поручения капитана, искать сокровища и зарабатывать уважение экипажа.
В зависимости от поступков игрока возможны несколько концовок с разной судьбой каждого из членов команды.

Примечание: сам мод LTM желателен, но не обязателен к установке.

Особенности:
- Полноценный пиратский лагерь;
- Уникальная внешность и проработанные характеры каждого из персонажей;
- Возможность попасть на борт Эсмеральды;
- Раб Харек сможет отдать долг герою, если тот поможет ему освободиться от рабства;
- Сибуру Нараду добавлена торговля;
- Полностью реализована и сбалансирована репутация у пиратскиой фракции.
