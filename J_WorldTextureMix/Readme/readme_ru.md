Данная модификация не совместима со следующими модами: QPTexture, MyrtanaTexturePack, PreAlpha, AutumnTextures, SmartTexturesPack, RetexturedFinal.

Этот мод добавляет очень реалистичные текстуры. Цветовая гамма мира основана на базовой игре. Начиная с этой версии, у меня есть возможность добавлять уникальные текстуры в любую точку игрового мира, что делает мод единственным с большим разнообразием. С каждым обновлением разнообразие мода будет увеличиваться.
