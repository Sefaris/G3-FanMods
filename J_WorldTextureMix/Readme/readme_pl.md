Ta modyfikacja nie jest kompatybilna z następującymi modami: QPTexture, MyrtanaTexturePack, PreAlpha, AutumnTextures, SmartTexturesPack, RetexturedFinal.

Ten mod dodaje bardzo realistyczne tekstury. Kolorystyka świata jest oparta na podstawowej grze. Od tej wersji mam możliwość dodania unikalnych tekstur w każdym miejscu w świecie gry, dzięki czemu mod, jako jedyny, jest bardzo zróżnicowany. Z każdą kolejną aktualizacją zróżnicowanie moda będzie się powiększać.
