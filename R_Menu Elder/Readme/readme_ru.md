Данная модификация не совместима со следующими модами: MenuCM, MenuOLD, MenuDark, MenuArcania, MenuQP, UpdatedGUI, Immersive World, CompasReturning

Примечание: в данном интерфейсе уже имеется компас, так что возвращать его нет смысла.
