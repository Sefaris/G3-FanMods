Базовая часть Контент Мода. Данная модификация необходима для работы остальных модов линейки. Изменения, которые будут заметны в игре с этим модом, полностью основаны на материалах оригинальной Готики 3 и дополнения Отвергнутые Боги.

Список изменений:
- Более разнообразная броня нордмарцев;
- Более разнообразная броня ассасинов;
- Более разнообразная броня наёмников;
- Женщины среди повстанцев и наёмников;
- Мантии для послушников Монастыря;
- Головные уборы и шлемы для НПС по всему миру;
- Более тёмный фон пергамента для писем и книг;
- Новые головы Лареса, Инога, Анога, Хавьера, Ватраса;
- Уникальные тела у главарей нордмарских кланов и друидов;
- Изменённый внешний вид орочьего доспеха;
- Уникальная внешность Кана;
- Новая внешность зомби;
- С мумий и проклятых жрецов падает лут;
- Более качественная текстура главного героя, голова в 4К;
- Новые имена стандартных НПС.
