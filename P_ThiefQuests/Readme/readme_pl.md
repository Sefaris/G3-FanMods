Wymagane: Main


Wprowadza zmiany w złodziejskich przygodach bohatera. Dodaje nowe zadania i modyfikuje część starych. Na przykład skarbiec Trelis jest zamknięty i aby go okraść będziesz musiał użyć jednej z umiejętności złodziei: kradzieży kieszonkowej lub ogłuszenia. Strażnicy skarbów w Geldern zareagują agresywnie na bohatera, jeśli spróbuje dostać się do środka. Zadanie ze złotymi talerzami dla Laresa nie będzie anulowane. Nowe zadania złodzieja w Ben Erai, Ishtar i Bakareshu. Za ukończenie wszystkich zadań złodziei w grze możesz zdobyć Wytrych mistrza złodziei, który może otworzyć prawie każdą skrzynię. Ale to nie znaczy, że można go łatwo zdobyć i zdobyć za pompowanie umiejętności otwierania zamków. Gracz będzie musiał się bardzo postarać i odwiedzić prawie wszystkie miasta w grze.
