A mod that creates the basis for installing subsequent mods. By itself, it adds some innovations from the LTM and Consequences mods.

Changes visible to the player:
- Mod rebalance.
- Changed assortment of some merchants and loot from some monsters.
- Increased size of the Vengard barrier, magic plates, port and statue of Innos in the temple from LTM.
- Changed beds throughout Myrtana.
- At the beginning of the game, all friends are in Ardea.
- A character named Tristan. Without the Consequences mod, there will be no dialogue with him.
- Locked doors in front of the captives Gorn, Lars and Wenzel.
- Wenzel needs to return his sword before he agrees to liberate Cape Dun.
