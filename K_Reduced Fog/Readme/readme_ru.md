Данная модификация в значительной степени уменьшает игровой туман, скрывающий дальние планы. По сути, происходит увеличение дальности отрисовки без потери производительности. Особенно сильно эффект заметен в Нордмаре.

Примечание: после установки данного мода необходимо обязательно перезагрузить файл ShaderCashe. Игра сама сделает это при первом запуске. Не пытайтесь доставать бэкап данного файла в папке Документы.
