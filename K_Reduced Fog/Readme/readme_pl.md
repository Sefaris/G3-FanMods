Ta modyfikacja znacznie zmniejsza mgłę w grze, która ukrywa odległe obszary. Zasadniczo zwiększa się odległość renderowania bez utraty wydajności. Efekt jest szczególnie zauważalny w Nordmarze.

Uwaga: po zainstalowaniu tego moda pamiętaj o ponownym załadowaniu pliku ShaderCashe. Sama gra zrobi to przy pierwszym uruchomieniu. Nie próbuj utworzyć kopii zapasowej tego pliku w folderze Dokumenty.
