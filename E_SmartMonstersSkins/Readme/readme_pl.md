Retexture zwierząt i potworów. Niektóre tekstury zostały ulepszone do rozdzielczości 4K, podczas gdy inne zostały całkowicie zaktualizowane. Zmieniono tekstury lwów, ognistych jaszczurów, nosorożców, zębów szablozębów, szkieletów, rozpruwaczy, goblinów, a nawet żółwi i chrząszczy.

Uwaga: instalując ten mod z ContentModBase, pamiętaj, że zęby szablozębów zmienią kolor na czerwony.
