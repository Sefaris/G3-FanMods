Требования: TotalRebalance, NewBalance

Адаптационный мод для TotalRebalance и NewBalance.
 
Особенности:
- Увеличенная стоимость навыков владения оружием 2го и 3го уровня
- Всё оружие и щиты теперь требуют только первый уровень владения оружием
- Ребаланс опыта за победу над НПС. Можно играть со 100% получаемого опыта . Максимальный урон мобов включен через ini файл. 
- Настроенный ini файл NewBalance
