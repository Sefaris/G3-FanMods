Requirements:TotalRebalance, NewBalance

An adaptation mod for TotalRebalance and newBalance.
 
Features:
- Increased cost of level 2 and 3 weapon proficiency
- All weapons and shields now require only the first level of weapon proficiency
- Rebalance of experience for defeating NPCs. You can play with 100% of the experience gained. The maximum damage of mobs is enabled via the ini file. 
- Configured newBalance ini file
