Требования: TotalRebalance, ShieldRebalance

Адаптационный мод для TotalRebalance и ShieldRebalance. С этим модом зачарованные щиты не дают бонус к защите. Вместо этого они с определённым шансом накладывают на врага эффект при блоке. За основу для молниевых щитов был взят эффект “Заточенный”, который с определённым шансом наносит врагу урон и ненадолго станит его. Также добавлены следующие возможности:
- После изучения заклинания “Благословление оружия” можно благословлять щиты с помощью специального предмета в инвентаре.
- Возможность отравить щиты с помощью специального яда.
