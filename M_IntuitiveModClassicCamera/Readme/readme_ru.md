В данной модификации представлены всего 2 изменения:
1) Движения героя в любую сторону (назад, вправо, влево, по диагонали) приравнены по скорости к движению вперёд;
2) Моментальный поворот героя вслед за камерой.

Примечание: данная модификация рекомендуется только для игры с классической камерой и плохо сочетается с альтернативной камерой.
Примечание 2: немного облегчен геймплей за лучника.
