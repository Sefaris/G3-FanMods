Ta modyfikacja wprowadza tylko 2 zmiany:
1) Ruchy bohatera w dowolnym kierunku (do tyłu, w prawo, w lewo, po przekątnej) mają taką samą prędkość jak ruch do przodu;
2) Natychmiastowy obrót bohatera wspólnie z kamerą.

Uwaga: Ta modyfikacja jest zalecana tylko do gry z klasyczną kamerą i nie łączy się dobrze z alternatywną kamerą.
Uwaga 2: Nieco ułatwia grę pod łuki.
