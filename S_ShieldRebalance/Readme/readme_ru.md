Требование: New Balance and Combat

Блок зачарованным щитом накладывает на противника соответствующий эффект:
- Ослабление накладывает эффект ослабления на противника
- Отравление накладывает эффект отравления на противника
- Поджёг поджигает противника
- Обморожение замораживает противника
- Заточка наносит противнику урон
- Благославдение наносит урон существам Белиара

Кроме того блок магическим посохом отталкивает заклинание обратно в противника в обмен на ману.
