Wymagane: New Balance and Combat, New Game Configuration


Mod zmienia ustawienia moda New Balance And Combat, nie zmieniając jego zawartości, a także zmienia ustawienia oryginalnego moda w kierunku zwiększenia realizmu dynamiki i komfortu walki. Automatyczne poziomowanie jest wyłączone. Łucznicy nie będą tak często trafiać gracza w ruchu. Osłabiono ogłuszenia npc na graczu i gracza na npc. Na początku gry bohater będzie miał 3 mikstury leczące osłabienie, aby poradzić sobie z jaszczurami w jaskini Reddock. I inne zmiany ustawień z moda New Balance And Combat.
