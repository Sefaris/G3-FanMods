Переработка и логическое продолжение такой же функции из мода ЛСТМ, в котором можно телепортировать спутников героя. В данной версии можно телепортировать не только четырёх главных друзей, но и многих других персонажей. Расширен список точек для телепортации. Соблюдён баланс, к примеру паладинов нельзя телепортировать в орочьи города, пока они не будут освобождены.
