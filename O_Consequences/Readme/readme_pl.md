Wymagane: Main

Modyfikacja dodaje do gry wydarzenia, które są powodowane przez działania bohatera, który może wydawać się „wrogi” jednej lub drugiej stronie konfliktu. Do gry dodano nowe dialogi, klimatyczne detale, nowe postacie i zadania.

Możesz wybrać poziom trudności moda w początkowym dialogu z Hamlarem lub Tristanem.
