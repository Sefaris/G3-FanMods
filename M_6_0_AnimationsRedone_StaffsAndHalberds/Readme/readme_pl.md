Mod dodaje nowe animacje dla kosturów, kopii i halabard. Modyfikacja zawiera kilka zestawów animacji. Zestaw animacji można dostosować samodzielnie, korzystając z poniższych modów lub użyć klasycznego.

Klasyczny zestaw animacji obejmuje:

•	Zwykły atak – uderzenia pionowe.
•	Szybki atak – uderzenia poziome.
•	Silny atak – pchnięcie.
•	Atak wykańczający – uderzenie pionowe.
•	Atak okrężny – seria szybkich uderzeń.
