Мод добавляет новые анимации для посохов, копий и алебард. Модификация включает в себя несколько наборов анимаций. Набор анимаций можно настроить самостоятельно используя моды ниже либо воспользоваться классическим.

Классический набор анимаций включает в себя:

•	Обычная атака - вертикальные удары
•	Быстрая атака - горизонтальные удары
•	Сильная атака - укол
•	Добивающая атака - вертикальный удар
•	Круговая атака - серия быстрых ударов
