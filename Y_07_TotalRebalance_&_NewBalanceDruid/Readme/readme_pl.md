Wymagane: TotalRebalance, NewBalance

Mod adaptacyjny dla TotalRebalance i NewBalance.
 
Cechy:
-Zwiększono koszt umiejętności broni 2. i 3. poziomu
-Wszystkie bronie i tarcze wymagają teraz tylko umiejętności 1. poziomu.
-Zrównoważone doświadczenie za pokonanie NPC. Można grać ze 100% zdobytego doświadczenia. Maksymalne obrażenia zadawane przez moby są ustawiane za pomocą pliku ini. 
-Niestandardowy plik ini NewBalance

Uwaga! Oryginalna modyfikacja NewBalance nadal zawiera błędy, przez które nie można atakować niektórych przeciwników w postaci zwierzęcia. 
Aby móc atakować w formie zwierzęcia, należy kliknąć kółko myszy po transformacji.
