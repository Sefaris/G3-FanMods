Требуется включенный альтернативный баланс в главном меню игры.

Теперь не нужно откладывать выполнение заданий и зелья с перманентными прибавками атрибутов на потом, используйте их сразу, как только сможете, игра за это больше не наказывает!
Плагин меняет систему ограничения в прокачке с порогом в 250 силы, ловкости и древнего знания. Имеются возможности гибкой настройки, нажмите кнопку «Открыть настройки скрипта», чтобы изменить значения, имеется опциональные: облегченный ( без порогов обучения ) и хардкорный ( с дополнительными порогами обучения ) режимы. По умолчанию порог как в Альтернативном балансе 250 атрибутов. Перманенты не учитываются, и это можно отключить.

Внимание! Мод имеет критический баг, связанный с восстановлением выносливости во время использования расходников!
