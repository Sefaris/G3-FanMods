Wymaga włączenia alternatywnego balansu w menu głównym gry.

Nie musisz już opóźniać używania bonusów stałych do momentu przekroczenia progu nauki. 
Wtyczka zmienia system ograniczeń w uczeniu z progiem 250 siły, zręczności i prastarej wiedzy. Dostępne są opcjonalne tryby: lekki (bez progów treningowych) i hardcore (z dodatkowymi progami treningowymi). Domyślny próg jak w Alternatywnym Balansie to 250 atrybutów. Bonusy stałe nie są liczone ale można to wyłączyć.
