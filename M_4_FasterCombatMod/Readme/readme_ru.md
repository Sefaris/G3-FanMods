Компиляция некоторых старых наработок ISC от Mac Hyder и NewAnimation от ZimGurd с небольшой отсебятиной.

Целью является: актуализировать атаки двуручного и одноручного оружия при сохранении ванильных анимаций для комфортной игры с AAI.

Для того, чтобы нанести колющий удар оружием с типом “посох” – сделать двойной щелчок ПКМ.

Использованы наработки от: Mac Hyder и ZimGurd
