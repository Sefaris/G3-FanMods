Требования: Main, UpdatePack 1.05

Данная модификация не совместима со следующими модами: Balance Ultimate; Balance Ultimate & Immersive World

Модификация меняет механику призыва и дает возможность призывать до 4х существ одновременно.

Изучив заклинание призыва, игрок получает камень души соответствующего существа. Во время призыва камень души расходуется и отнимает  у игрока определенное количество максимальной маны. Игрок не может призвать существо повторно, пока не вернет камень души обратно. Сделать это можно двумя способами. Отозвать призванное существо либо забрать с его трупа. Вместе с камнем души игроку возвращается максимальная мана. Игрок может использовать все 4ре камня души и призвать одновременно гоблина, голема, скелета и демона

Внимание! В случае потери существа или его трупа вернуть камень души другими способами будет невозможно!

Также мод добавляет новый навык "Призыватель". После его изучения все камни души становятся универсальными, и игрок может использовать их для призыва любых существ. То есть можно будет вызвать несколько одинаковых существ при наличии камней души и выученного заклинания.
