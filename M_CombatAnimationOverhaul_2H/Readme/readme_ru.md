Мод полностью меняет боёвку на двуручных мечах и топорах и пытается сделать сражение с этими видами оружия менее неудобным и даже приятным

Этот мод включает в себя:
- Новые боевые анимации для двуручных мечей и топоров для людей, теперь обе расы используют свой собственный набор анимаций -> Значительно улучшен охват и плавность взмахов
- Новая блокирующая стойка для орков, чтобы лучше показывать, когда они блокируют
- Адаптированные хитбоксы для всего модифицированного оружия (Оружие из оригинальной игры. Не совместимо с балансными модами)
