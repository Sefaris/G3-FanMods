- Новый навык: Переписывание свитков. Нужен письменный стенд.
- Новый навык: Выплавка металла. Нужен кузнечный горн.
- Новый навык: Ювелирное дело. Нужен ювелирный верстак.
- Новые животные: бараны, овцы, петухи, верблюды, лисы.
- У некоторых домами появились погреба.
- Рыбылка
- Восстановлены недоделанные локации: Туннель между Миртаной и Нордмаром, проход в горах Нордмара, пещера возле Сильдене.

- Финальные слайды, отображается на весь экран.
- Черный фон при переходе в погреб отображается на весь экран.
- Обновление игрового интерфейса: добавлена дополнительная ячейка в горизонтали (раньше помещалось 5 предметов в ряд, теперь их 6).
