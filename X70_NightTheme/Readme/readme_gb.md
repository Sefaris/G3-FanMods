Version: 1.1.1 [2024.10.16]
Recommended requirements: UpdatePack v1.05.10 Beta
Modification not compatible with: any other GUI
Description: The modification is a reimagination of the game GUI designed for avid fans of dark themes, as well as inspired by the game "Gothic 2: Night of the Raven" and the long-term mod "Community Story Project".
Features:
•	The standard background images for the main menu and the loading screen have been replaced with new ones;
•	The area of the game interface has been expanded by X to the maximum 1024px (previously empty pixels at the edges of the screen are involved);
•	The number of columns of slots in the inventory has been increased from 5 to 6;
•	The layout of skill windows and spells has been changed to CSP style; the maximum number of slots has been increased in some sections;
•	The location of the status bars has been changed: they are now centered and located exactly under the quick access toolbar to always be in view (Risen style);
•	The mini-slides have been removed from the boot screen, and the position of the progress bar has been changed;
•	Updated the frames for pictographs of political factions and enclaves;
•	Updated lockpicking icons (UpdatePack v1.05.10 Beta);
•	The epilogue slides have been edited for widescreen monitors;
•	The display of empty skill/spell slots is disabled.
•	DDS collages have been removed; the game now reads individual icons instead of sets, which makes it easier to customize them with mods/mods.
•	Redesign of Gothic 3 icons to improve the quality and remove the blue outline.
•	New icons for transformation potions, their recipes and druid stones have been designed.
•	Mission items (ancestor stones and various kinds of supplies) now uses previously unused personal icons.
•	Selective editing of icons from modifications has been done to improve the quality of the most compressed TGA.
•	A !Customize! folder has been added, which contains files for manually replacing/customizing icons to the player's preferences.
•	Mod compatible with the Immersive World.
