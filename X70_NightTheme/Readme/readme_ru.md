Версия: 1.1.1 [2024.10.16]
Рекомендуемые требования: UpdatePack v1.05.10 Beta
Данная модификация несовместима со следующими модами: любой GUI-мод
Описание: Модификация является переосмыслением игрового интерфейса, предназначенным для заядлых любителей темных тем, а также вдохновленным игрой «Готика 2: Ночь Ворона» и модом-долгостроем «Community Story Project».
Особенности модификации:
•	Стандартные фоновые изображения для главного меню и экрана загрузки заменены на новые;
•	Область игрового интерфейса расширена по Х до предельных 1024px (задействованы ранее пустые пиксели по краям экрана);
•	Количество колонок слотов в инвентаре увеличено c 5 до 6;
•	Изменена разметка окон навыков (в стиле CSP) и заклинаний; в некоторых разделах увеличено максимальное количество слотов;
•	Изменено расположение строк состояния: теперь они центрированы и находятся точно под панелью быстрого доступа, чтобы всегда быть в поле зрения;
•	С загрузочного экрана удалены мини-слайды, а также изменено положение индикатора прогресса;
•	Обновлены рамки для пиктограмм политических фракций и анклавов;
•	Обновлены пиктограммы для взлома замков (UpdatePack v1.05.10 Beta);
•	Отредактированы слайды эпилога под широкоформатные мониторы;
•	Отключено отображение пустых слотов навыков/заклинаний.
•	Удалены dds-коллажи; игра теперь считывает отдельные иконки вместо наборов, что упрощает их кастомизацию модами/под моды.
•	Переработка иконок Gothic 3 с целью улучшения качества и удаления голубой обводки.
•	Оформлены новые иконки зельям превращения, их рецептам и камням друидов.
•	Квестовым предметам (камням предков и разного рода поставкам) прописаны ранее не использовавшиеся персональные иконки.
•	Проделана избирательная редактура иконок из модификаций с целью улучшения качества наиболее пережатых TGA.
•	Добавлена папка !Customize!, содержащая файлы для ручной замены/кастомизации иконок под предпочтения игрока.
•	Соблюдена совместимость с модификацией Immersive World.
