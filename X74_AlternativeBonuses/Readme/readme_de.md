Beschreibung: Die Modifikation überarbeitet die permanenten Boni, die im Originalspiel durch das Abschließen bestimmter Quests erreicht wurden. Die Änderungen sind vor allem für Perfektionisten gedacht, die beim Spielen mit Alternatives Balancing gezwungen waren, bestimmte Quests bis zu dem Zeitpunkt zu verschieben, an dem das Zielattribut auf 250+ gepumpt wird. Das ist nun nicht mehr nötig: Die Modifikation bietet fünf Modi, von denen einer durch das alte Tagebuch des Helden aktiviert wird, das im Inventar liegt.
Standard: Questboni werden durch Steintafeln ersetzt, die in "Gothic 2: Nacht der Raben" bekannt sind. Wenn bestimmte Bedingungen erfüllt sind, können sie von einigen Artefakthändlern über einen speziellen Dialog gekauft werden;
Alternativ: Lehrer mit bestimmten Talenten bieten zusätzliches Training, das es Ihnen ermöglicht, eine begrenzte Anzahl von Attributseinheiten für Gold zu erhöhen, ohne Trainingspunkte zu investieren;
Klassisch: Boni werden für Quests vergeben, die Hauptfunktionen des Mods sind ausgeschaltet. Startet automatisch, wenn Sie den Dialog-Aktivator verpassen;
Rational: für das Abschließen von Quests erhält der Held Lernpunkte anstelle von Attributpunkten;
Hardcore: Boni für Quests werden aus dem Spiel entfernt; die Funktionen der Modi Standard, Alternativ und Klassisch werden nicht verwendet.
