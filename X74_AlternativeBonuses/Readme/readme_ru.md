Версия: 1.1.1 [2024.10.16]
Поддерживаемые языки: English, Deutsch, Polski, Русский, Magyar
Описание: модификация перерабатывает получение игроком перманентных бонусов, которые в оригинальной игре достигались путем выполнения определенных квестов. Главным образом изменения предназначены для перфекционистов, которые при игре с Альтернативным Балансом были вынуждены откладывать те или иные квесты до времен, когда целевой атрибут будет прокачан до 250+. Теперь в этом необходимости нет: Модификация предлагает пять режимов, один из которых активируется через лежащий в инвентаре старый дневник Героя.
Стандартный: бонусы с квестов заменяются каменными табличками, знакомыми по "Готике 2: Ночь Ворона". При соблюдении некоторых условий они могут быть куплены у некоторых торговцев артефактами через особый диалог;
Альтернативный: у учителей при наличии у героя определенных талантов появляется дополнительное обучение, которое позволит за золото поднять ограниченное количество единиц атрибута без вложений очков обучения;
Классический: бонусы начисляются за квесты, основные функции мода выключены. Запускается автоматически, если пропущен активатор;
Рациональный: за квесты вместо бесплатных атрибутов начисляются очки обучения, которые герой может распределять по своему усмотрению;
Хардкорный: из игры удалены бонусы за квесты; функции остальных режимов не задействованы.
