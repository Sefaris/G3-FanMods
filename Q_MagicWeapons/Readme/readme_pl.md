Wymagane: Main

Na całym świecie czarodzieje mogą zdobyć nowe magiczne bronie z charakterystycznymi obrażeniami od magii. Dla magów ognia - kostury ognia, dla magów wody - kostury lodu, dla mrocznych magów - kostury elektryczności. W oryginalnej grze atrybut "Ochrona przed energią" jest bezużyteczny ze względu na brak wrogów zadających obrażenia od energii.

Przepisy na nowe magiczne bronie zostały rozmieszczone po całym świecie.
