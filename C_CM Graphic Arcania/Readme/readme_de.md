Voraussetzungen: Content Mod Base

Diese Modifikation fuegt der ContentMod grafische Aenderungen hinzu, die von Risen und Arcania übernommen wurden.
Änderungen:
- Neue Koepfe aus Arcania (Lester, Diego, Milten, Gorn, Thorus, Xardas, Bufford, Ortega, Marcus, Kobrin, Roland, Grim)
- Einige Banditen tragen Banditenruestungen aus Risen
- Einige Rebellen und freigelassene Sklaven sind mit Kleidung aus Risen bekleidet
- Einige NPCs tragen Jagdkleidung aus Arcania
