Requirement: Content Mod Base

This modification adds graphical changes for Content Mod borrowed from Risen and Arcania.

Changes:
- New heads from Arcania (Lester, Diego, Milten, Gorn, Thorus, Xardas, Bufford, Ortega, Markus, Cobrin, Roland, Grim);
- Some bandits are dressed in bandit armor from Risen;
- Some rebels and freed slaves are dressed in clothes from Risen;
- Some NPCs are dressed in hunting clothes from Arcania.
