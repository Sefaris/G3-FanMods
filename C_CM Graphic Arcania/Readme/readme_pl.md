Wymagane: Content Mod Base

Ta modyfikacja dodaje zmiany graficzne do ContentMod zapożyczone z Risen i Arcania.

Zmiany:
- Nowe głowy z Arcanii (Lester, Diego, Milten, Gorn, Thorus, Xardas, Bufford, Ortega, Marcus, Kobrin, Roland, Grim);
- Niektórzy bandyci są ubrani w zbroje bandyckie z Risena;
- Niektórzy rebelianci i uwolnieni niewolnicy są ubrani w odzież z Risena;
- Niektóre postacie niezależne w strojach myśliwskich z Arcanii.
