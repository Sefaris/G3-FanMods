Требования: Content Mod Base

Данная модификация добавляет графические изменения КонтентМода, заимствованные из Risen и Arcania.

Изменения:
- Новые головы из Аркании (Лестер, Диего, Мильтен, Горн, Торус, Ксардас, Буффорд, Ортега, Маркус, Кобрин, Роланд, Грим);
- Некоторые бандиты наряжены в бандитский доспех из Ризена;
- Некоторые повстанцы и освобождённые рабы наряжены в одежду из Ризена;
- Некоторые НПС в охотничьих одеждах из Аркании.
