Mit dieser Modifikation wird das Aussehen fast aller Woelfe im Spiel neu gestaltet. 
Die Modelle sind von Risen und Arcania übernommen. 
Neues Aussehen der normalen Woelfe, der Wargs, der hungrigen Woelfe und der Eiswoelfe. 
Die Trophaeen an den Wänden wurden ebenfalls geaendert, um dem neuen Modell zu entsprechen.
