Wersja: 1.1.1 [2024.10.16]
Obsługiwane języki: Angielski, Niemiecki, Polski, Rosyjski, Węgierski
Opis: Ta modyfikacja zmienia mechanikę kradzieży kieszonkowej, czyniąc ją podobną do jej odpowiedników w "Gothic 2: Noc Kruka" i "Kronik Myrtany: Archolos". Wszystkie nazwane postacie niezależne mają spersonalizowane skrypty kradzieży i unikalne łupy; szansa na porażkę jest całkowicie wyeliminowana: bohater może albo opróżnić kieszenie swojej ofiary, albo nie.
