Требования: CM Rebalance; Intuitive Combat System; Pirate Dream; ExtendedFrying; New Balance and Combat; Consequenes; LTM Easy

Данная модификация не совместима со следующими модами: CNQDragons; Total Rebalance; Balance Ultimate; CNQ Arenas; Alternative Bonuses; CNQMagicWeapons

Модификация позволяет играть со многими современными модификациями, сохраняя при этом умеренную сложность игры.

Некоторые особенности:
- Отключен выбор последствий, по умолчанию выбирается лайт-версия
- Перебалансированы анимации каста заклинаний, добавлены новые анимации
- Замедлены некоторые боевые анимации падальщиков и волков
- Перебалансирован лут в сундуках
- Новые секреты и нычки
- Понижены уровни всех магов нежити в игре
