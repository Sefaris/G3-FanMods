Требования: Main

Данный мод добавляет в игру новую сюжетную ветку, построенную вокруг мести нашего знакомого Ли королю Робару. Сюжет задействует большую часть населения Венгарда, а так же персонажей из Нордмара, и приведет к трем различным концовкам в зависимости от выбора фракций.
Данная версия мода - ЛТМовская. Исправлены многочисленные баги и недоработки. Если отсутствует желание проходить квестовую цепочку, можно выбрать ванильный вариант в диалоге с ЛИ.
