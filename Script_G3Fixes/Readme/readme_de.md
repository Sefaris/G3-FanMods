Dieses Skript behebt einige Spielmechanismen und fügt neue hinzu.
- Neue Regeln für Statuseffekte bei Zaubern, magischen Waffen und Geschossen
- Die Gefährten des Helden können sofort auf unfreundliche Kreaturen reagieren.
- Geheimes Verhalten von Monstern: Wenn der Spieler ein Monster anlockt, kann er die ganze Meute zu sich locken.
- Konsolenbefehl zur Wiederbelebung von NPCs hinzugefügt: resetpos NPC_INSTANCE
Hinzufügen der Möglichkeit, Wasserfall-Sounds zu deaktivieren
- Hinzufügen der Möglichkeit, das Respawning von Monstern zu deaktivieren
