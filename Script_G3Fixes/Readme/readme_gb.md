This script fixes some game mechanics, and adds new ones.
- New rules for imposing status effects on spells, magic weapons and projectiles
- Hero's companions can instantly react to unfriendly creatures
- Secret behavior of monsters: by luring one monster, the player can lure the whole pack to himself.
- Added console command to revive NPCs: resetpos NPC_INSTANCE
- Added the ability to disable waterfall sounds
- Added ability to disable monster respawning
