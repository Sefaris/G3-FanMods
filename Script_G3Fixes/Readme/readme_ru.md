Этот скрипт исправляет некоторые игровые механики, а также добавляет новые.
- Новые правила наложения статусных эффектов у заклинаний, магического оружия и снарядов
- Спутники героя могут мгновенно реагировать на недружественных существ
- Стайное поведение монстров: выманивая одного монстра, игрок может сагрить на себя всю стаю
- Добавлена консольная команда для оживления НПС: resetpos NPC_INSTANCE
- Добавлена возможность отключения звуков водопадов
- Добавлена возможность отключить респавн монстров
