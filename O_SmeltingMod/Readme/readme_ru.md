Требования: Main

Продолжение идеи других модов по интерактивному взаимодействиую с кузнецами. Можно переплавлять на кузнечные заготовки сырую руду и ржавое оружие. Переплавкой можно заниматься у любого кузнеца, либо в плавильной печи. Требования и стоимость заготовок у всех кузнецов и у плавильной печи одинаковая. В печи также можно переплавлять золото и руду.
