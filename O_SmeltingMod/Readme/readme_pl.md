Wymagane: Main

Kontynuacja idei innych modów do interaktywnej interakcji z kowalami. Możesz przetopić surową rudę i zardzewiałą broń na sztabki. Przetapianie można wykonać u dowolnego kowala lub w piecu do wytapiania. Wymagania i koszt sztabek są takie same dla wszystkich kowali i pieca. Piec może również wytapiać złoto i rudę.
Zawiera również zadanie LTM dotyczące sztabek złota jako prezentu dla strażnika Ishtar w celu zwiększenia reputacji wśród asasynów.
